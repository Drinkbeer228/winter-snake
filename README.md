# ❄️ Зимняя Змейка - Arctic Snake

## 🎮 Что это за игра?

**Зимняя Змейка** - это современная переосмысленная классическая игра "Змейка" с зимней атмосферой, системой уровней и дофаминовой петлей удержания. Создана с нуля на модульной JavaScript архитектуре для максимальной производительности и масштабируемости.

## 🎯 Ключевые фичи

### 🌊 **5 Уровней Прогрессии**
- **❄️ Уровень 1: Зимний лес** (0-99 очков) - Классический геймплей
- **🌨 Уровень 2: Снежная буря** (100-199) - Еда двигается, скорость +20%
- **🧊 Уровень 3: Ледяные блоки** (200-299) - Препятствия на поле
- **🌫 Уровень 4: Туман** (300-399) - Ограниченная видимость 5 клеток
- **🌌 Уровень 5: Северное сияние** (400+) - Легендарный режим с аурой

### ⚡ **Система Прогрессии**
Каждые 100 очков игрок получает новую механику и визуальный эффект, что создаёт постоянное чувство прогресса.

### 🎵 **Аудио-система**
- **Звуковые эффекты** - синтезированные через Web Audio API
- **Фоновая музыка** - "Never mind 1L23" by Setuniman
- **Раздельные каналы** - музыка не мешает звукам игры

### 🎨 **Визуальные Эффекты**
- **2.5D графика** - тени, градиенты, объем
- **Частицы** - эффекты при поедании еды
- **Динамический фон** - меняется с уровнями
- **Туман войны** - реалистичный эффект ограниченной видимости
- **Северное сияние** - радужная аура на высшем уровне

### 🎮 **Управление**
- **Клавиатура** - стрелки, пробел (пауза), ESC (меню паузы)
- **Тач-экран** - свайпы для мобильных устройств
- **Мобильные кнопки** - джойстик как запасной вариант
- **Адаптивный интерфейс** - работает на всех устройствах

### 📱 **Мобильная Совместимость**
- **iOS оптимизация** - специальные обработчики тач-событий
- **Android поддержка** - стандартные тач-события
- **Адаптивный дизайн** - подстраивается под размер экрана
- **Предотвращение скролла** - игра не "прыгает" при свайпах

## 🏗️ Технический Стек

### 📁 **Модульная Архитектура**
```
snake/
├── index.html          # Чистый HTML5
├── css/style.css       # Стили с анимациями
├── js/
│   ├── config.js       # Конфигурация уровней
│   ├── state.js        # Состояние игры
│   ├── audio.js        # Звуки + музыка
│   ├── render.js       # Отрисовка канваса
│   ├── game.js         # Логика игры
│   └── main.js         # Инициализация
└── assets/music/       # Для будущих треков
```

### 🛠️ **Технологии**
- **Vanilla JavaScript ES6+** - без фреймворков
- **Canvas 2D API** - высокопроизводительная графика
- **Web Audio API** - синтез звуков
- **LocalStorage** - сохранение прогресса
- **CSS3 Animations** - плавные переходы

## 🎮 Геймплей Механики

### 🐍 **База**
- Классическое управление змейкой (стрелки/тач)
- Поедание еды для роста
- Избегание стен и самого себя

### 🚀 **Продвинутые механики**
- **Комбо-система** - множитель очков за быстрое поедание
- **Движущаяся еда** - усложняет на высоких уровнях
- **Препятствия** - ледяные блоки на поле
- **Туман** - ограниченная видимость для хардкора
- **Бонусная еда** - золотые снежинки с таймером

### 📊 **Прогрессия**
- **Очки** - базовая валюта
- **Уровни** - открывают новые механики
- **Скорость** - растёт с прогрессом
- **Сложность** - адаптивная, не ломает игру

## 🚀 Почему эта игра заиграет?

### 🎮 **Геймплей**
- **Знакомые правила** - игроки сразу понимают что делать
- **Плавная кривая сложности** - не отпугивает новичков
- **Постоянный прогресс** - каждый уровень даёт что-то новое

### 💊 **Доффамин**
- **Частые вознаграждения** - каждые 100 очков
- **Визуальная обратная связь** - эффекты, переходы, звуки
- **Чувство роста** - от новичка до легенды

### 🎨 **Атмосфера**
- **Зимняя тема** - уютная и успокаивающая
- **Минимализм** - не перегружает внимание
- **Плавность** - всё анимировано, ничего не рвётся

### 📱 **Универсальность**
- **PC/Mobile** - работает везде
- **Быстрая загрузка** - модули оптимизированы
- **Отзывчивый дизайн** - адаптируется под экран
