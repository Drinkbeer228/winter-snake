# 🐍 Snake Game - Модульная версия

Классическая игра "Змейка" с современной архитектурой на ES6 модулях. Чистый JavaScript без зависимостей.

![Snake Game](https://img.shields.io/badge/Game-Snake-brightgreen) ![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow) ![License](https://img.shields.io/badge/License-MIT-blue)

## 🎮 Демо

Игра работает на всех устройствах:
- **Десктоп:** Управление стрелками на клавиатуре
- **Мобильные:** Свайпы по экрану
- **Планшеты:** Поддержка тач-управления

## 🏗️ Архитектура

### Структура файлов
```
js/
├── core/
│   ├── Game.js         - главный класс-дирижёр
│   ├── Snake.js        - класс змейки
│   ├── Renderer.js     - класс рендерера
│   └── Input.js        - класс управления
├── utils/
│   ├── config.js       - конфигурация и константы
│   ├── state.js        - глобальное состояние игры
│   └── audio.js        - простая заглушка звука
└── main.js            - точка входа
```

### Принципы
- **ES6 Modules** - import/export без сборщиков
- **Один файл = одна сущность** - логическое разделение
- **Минимум зависимостей** - чистый нативный JS
- **Расширяемость** - легко добавлять новые функции

## 🚀 Быстрый старт

### Клонирование и запуск
```bash
git clone https://github.com/Drinkbeer228/winter-snake.git
cd winter-snake
python -m http.server 5500
```

### Открыть игру
```
http://localhost:5500
```

**⚠️ Важно:** Не открывайте `index.html` напрямую через `file://` - ES6 модули работают только через HTTP-протокол.

### Альтернативные способы запуска
```bash
# Использование Node.js
npx serve .

# Использование PHP

# Использование Live Server (VS Code)
# Установите расширение и кликните правой кнопкой на index.html
```

## 🔧 Конфигурация

Файл `js/utils/config.js` содержит основные настройки:

```javascript
export const CONFIG = {
  GRID: 20,              // Размер сетки (клеток)
  CANVAS_SIZE: 400,      // Размер поля (пиксели)
  INITIAL_SPEED: 400,    // Начальная скорость (мс)
  MIN_SPEED: 50,         // Минимальная скорость (мс)
  SPEED_INCREMENT: 5,    // Увеличение скорости за еду
  soundEnabled: false,   // Звук включен/выключен
  debug: true           // Режим отладки
};
```

### Настройка цветов
```javascript
export const COLORS = {
  BACKGROUND: '#111',   // Фон
  SNAKE: '#0f0',        // Змейка
  FOOD: '#f00',         // Еда
  GRID: '#333'          // Сетка
};
```

## 🎯 Как это работает

### Game.js - дирижёр
- Управляет игровым циклом
- Вызывает `update()` и `render()`
- Обрабатывает коллизии и еду

### Snake.js - змейка
- Хранит массив сегментов
- Двигается, растёт, проверяет коллизии
- Обрабатывает изменение направления

### Renderer.js - отрисовка
- Очищает canvas
- Рисует сетку, змейку, еду
- Никакой логики - только отрисовка

### Input.js - управление
- Слушает клавиатуру и тач
- Вызывает колбэк при изменении направления
- Конвертирует свайпы в направления

### Структура кода
- **Game.js** - главный игровой цикл и логика
- **Snake.js** - класс змейки с движением и ростом
- **Renderer.js** - отрисовка на canvas
- **Input.js** - обработка клавиатуры и тач-событий
- **config.js** - все настройки игры
- **state.js** - глобальное состояние

## 📦 Добавление новых функций

### Новая механика
1. Добавить в `Game.js` в метод `update()`
2. Добавить состояние в `state.js`
3. Добавить отрисовку в `Renderer.js`

### Новые настройки
1. Добавить в `CONFIG` в `config.js`
2. Использовать в нужных модулях через импорт

### Новый UI
1. Создать файл в `js/ui/`
2. Импортировать в `main.js`
3. Инициализировать в `initGame()`

## 🎨 Кастомизация

### Изменение внешнего вида
- **Цвета:** Изменить `COLORS` в `config.js`
- **Размеры:** Модифицировать `CONFIG.GRID` и `CONFIG.CANVAS_SIZE`
- **Скорость:** Настроить `INITIAL_SPEED` и `SPEED_INCREMENT`

### Добавление тем
```javascript
// В config.js
export const THEMES = {
  classic: { SNAKE: '#0f0', FOOD: '#f00', BACKGROUND: '#111' },
  ocean: { SNAKE: '#00f', FOOD: '#ff0', BACKGROUND: '#001' },
  sunset: { SNAKE: '#f80', FOOD: '#08f', BACKGROUND: '#210' }
};
```
