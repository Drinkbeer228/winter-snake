# Snake Game - Модульная версия

Минимальная реализация классической змейки с модульной архитектурой на ES6.

## 🏗️ Архитектура

### Структура файлов
```
js/
├── core/
│   ├── Game.js         - главный класс-дирижёр
│   ├── Snake.js        - класс змейки
│   ├── Renderer.js     - класс рендерера
│   └── Input.js        - класс управления
├── utils/
│   ├── config.js       - конфигурация и константы
│   ├── state.js        - глобальное состояние игры
│   └── audio.js        - простая заглушка звука
└── main.js            - точка входа
```

### Принципы
- **ES6 Modules** - import/export без сборщиков
- **Один файл = одна сущность** - логическое разделение
- **Минимум зависимостей** - чистый нативный JS
- **Расширяемость** - легко добавлять новые функции

## 🚀 Запуск

### 1. Локальный сервер
```bash
python -m http.server 5500
# или
npx serve .
```

### 2. Открыть в браузере
```
http://localhost:5500
```

**Важно:** Не открывать `index.html` напрямую через `file://` - ES6 модули работают только через HTTP.

## 🎮 Управление

- **Клавиатура:** Стрелки ↑↓←→
- **Мобильный:** Свайпы в любом направлении
- **Перезапуск:** F5 после Game Over

## 🔧 Конфигурация

Файл `js/utils/config.js` содержит основные настройки:

```javascript
export const CONFIG = {
  GRID: 20,              // Размер сетки
  CANVAS_SIZE: 400,      // Размер поля
  INITIAL_SPEED: 400,    // Начальная скорость (мс)
  MIN_SPEED: 50,         // Минимальная скорость
  soundEnabled: false,   // Звук включен/выключен
  debug: true           // Режим отладки
};
```

## 🎯 Как это работает

### Game.js - дирижёр
- Управляет игровым циклом
- Вызывает `update()` и `render()`
- Обрабатывает коллизии и еду

### Snake.js - змейка
- Хранит массив сегментов
- Двигается, растёт, проверяет коллизии
- Обрабатывает изменение направления

### Renderer.js - отрисовка
- Очищает canvas
- Рисует сетку, змейку, еду
- Никакой логики - только отрисовка

### Input.js - управление
- Слушает клавиатуру и тач
- Вызывает колбэк при изменении направления
- Конвертирует свайпы в направления

## 🐛 Отладка

При `CONFIG.debug = true` в консоли доступен:
```javascript
window.__game
// __game.state.score - текущий счёт
// __game.snake.segments - сегменты змейки
// __game.gameOver() - принудительный Game Over
```

## 📦 Добавление новых функций

### Новая механика
1. Добавить в `Game.js` в метод `update()`
2. Добавить состояние в `state.js`
3. Добавить отрисовку в `Renderer.js`

### Новые настройки
1. Добавить в `CONFIG` в `config.js`
2. Использовать в нужных модулях через импорт

### Новый UI
1. Создать файл в `js/ui/`
2. Импортировать в `main.js`
3. Инициализировать в `initGame()`

## 🎨 Стили

Базовые стили в `index.html`. Для кастомизации:
- Изменить `COLORS` в `config.js`
- Модифицировать методы в `Renderer.js`
- Добавить CSS классы в `index.html`
