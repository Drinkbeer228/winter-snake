# 🐍 Winter Snake — Модульная версия

Классическая игра "Змейка" с современной архитектурой на ES6 модулях. Чистый JavaScript без зависимостей.

---

## 🎮 Демо

Открой `index.html` в браузере или запустите локальный сервер:
```bash
python -m http.server 5500
# Открыть http://localhost:5500
```

---

## 🎯 Особенности игры

### Геймплей
- Классическая механика змейки с ростом при поедании еды
- Прогрессивная сложность - препятствия появляются каждые 50 очков
- Автоматическая генерация еды в случайных местах
- Collision detection для стен, собственного тела и препятствий
- Ускорение при нажатии направления (500мс буст)

### Технические особенности
- Canvas rendering для плавной анимации
- Адаптивный дизайн - автоматически подстраивается под экран
- Touch support для мобильных устройств
- Модульная архитектура на ES6
- Поддержка спрайтов и emoji

## 🎯 Как это работает

### Game.js - дирижёр
- Управляет игровым циклом
- Вызывает update() и render()
- Обрабатывает коллизии и еду
- Адаптивное масштабирование

### Snake.js - змейка
- Хранит массив сегментов
- Двигается, растёт, проверяет коллизии
- Обрабатывает изменение направления

### Renderer.js - отрисовка
- Очищает canvas
- Рисует сетку, змейку, еду
- Поддержка спрайтов и emoji
- Никакой логики - только отрисовка

### Input.js - управление
- Слушает клавиатуру и тач
- Вызывает колбэк при изменении направления
- Конвертирует свайпы в направления

## 🎨 Кастомизация

### Изменение внешнего вида
```javascript
// config.js
COLORS: {
  snakeHead: '#ff6b6b',  // Новый цвет головы
  snakeBody: '#4ecdc4'   // Новый цвет тела
}
```

### Добавление тем
```javascript
// Добавить в config.js
THEMES: {
  dark: { /* цвета */ },
  light: { /* цвета */ }
}
```

### Замена спрайтов
1. Положи спрайты в `assets/images/`
2. Обнови пути в `js/core/Renderer.js`
3. Размер спрайтов: 32×32px
